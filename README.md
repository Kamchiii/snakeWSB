# Snake
Gra w repozytorium to gra snake która umożliwia granie w trybie jednoosobowym lub dwuosobowym

### Uruchomienie gry
żeby uruchomić grę potrzebujemy 
```bash
git clone https://github.com/Kamchiii/snakeWSB.git
cd SnakeGame
dotnet run
```

Sterowanie 
Akcja | Gracz 1 | Gracz 2
--- | --- | ---
Góra | W | strzałka w górę
Dół | S | strzałka w dół
Lewo | A | strzałka w lewo
Prawo | D | strzałka w prawo

### Zasady gry
Wąż automatycznie idzie do przodu, sterując nim zbieraj gwiazdki żeby rosnąć
wygrywa osoba z największym wynikiem lub ostatnia przy życiu
